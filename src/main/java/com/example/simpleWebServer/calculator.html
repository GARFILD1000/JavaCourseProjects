Really cool calculator for real mens!

Simple rules for calculator:

  • In an expression, all operators and numbers must be separated by spaces
  • The integer and fractional parts must be separated by a point
  • The priority of operations is the same as in ordinary arithmetic
  • Example of right expression: 52 - 2.14 + 3.14 - ( 27 / ( 3 + 5 ) ) ^ 2
